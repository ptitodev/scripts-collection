Get-VsanIscsiTarget

Synopsis

This cmdlet retrieves vSAN iSCSI targets from clusters.

Syntax

Get-VsanIscsiTarget [-Cluster <Cluster[]>] [[-Name] <String[]>] [-IscsiQualifiedName <String[]>] [-Server <VIServer[]>] [<CommonParameters>]

Get-VsanIscsiTarget -Id <String[]> [-Server <VIServer[]>] [<CommonParameters>]

Related Commands

Online version
New-VsanIscsiLun
Remove-VsanIscsiLun
Set-VsanIscsiLun

Detailed Description

This cmdlet retrieves vSAN iSCSI targets from clusters. The cmdlet can filter targets by IQN and aliases.

Parameters

NameTypeDescriptionRequired?Pipeline InputDefault Value
NameString[]Filters the retrieved vSAN iSCSI targets by name.falsefalse
ClusterCluster[]Specifies the clusters from which to retrieve vSAN iSCSI targets.falsetrue (ByValue)
IdString[]Filters the retrieved vSAN iSCSI targets by ID.truefalse
IscsiQualifiedNameString[]Filters the retrieved vSAN iSCSI targets by IQN.falsefalse
ServerVIServer[]Specifies the vCenter Server systems on which you want to run the cmdlet. If no value is given to this parameter, the command runs on the default servers. For more information about default servers, see the description of Connect-VIServer.falsefalse

Return Type

Zero or more VsanIscsiTarget objects

Notes

Examples

-------------- Example 1 --------------

Get-VsanIscsiTarget -Cluster "vsan-cluster"

Retrieves all vSAN iSCSI targets from the "vsan-cluster" cluster.

-------------- Example 2 --------------

Get-VsanIscsiTarget -Cluster $cluster -Name "Target1"

Retrieves the vSAN iSCSI target named "Target1" from the $cluster vSAN cluster.
