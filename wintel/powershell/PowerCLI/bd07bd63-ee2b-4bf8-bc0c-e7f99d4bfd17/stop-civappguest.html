Stop-CIVAppGuest

Synopsis

This cmdlet shuts down the guest operating systems of all virtual machines in the specified vApp.

Syntax

Stop-CIVAppGuest [-VApp] <CIVApp[]> [-RunAsync] [-Server <CIServer[]>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-PipelineVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]

Related Commands

Online version
Restart-CIVAppGuest

Detailed Description

This cmdlet shuts down the guest operating systems of all virtual machines in the specified vApp.

Notes

Examples

-------------- Example 1 --------------

$myVApp = Get-CIVApp -Name 'MyVApp'
Stop-CIVAppGuest $myVApp

Shuts down the guest operating systems of all virtual machines in the specified vApp.

-------------- Example 2 --------------

$myVApp = Get-CIVApp -Name 'MyVApp'
$task = $myVApp | Stop-CIVAppGuest -RunAsync

Shuts down the guest operating systems of all virtual machines in the specified vApp asynchronously.
