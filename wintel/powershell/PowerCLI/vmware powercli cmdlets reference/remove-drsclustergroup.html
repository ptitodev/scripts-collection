Remove-DrsClusterGroup

Synopsis

This cmdlet removes the specified DRS cluster groups from the cluster on which it resides.

Syntax

Remove-DrsClusterGroup [-DrsClusterGroup] <DrsClusterGroup[]> [-RunAsync] [-Server <VIServer[]>] [-WhatIf] [-Confirm] [<CommonParameters>]

Related Commands

Online version
Get-DrsClusterGroup
New-DrsClusterGroup

Detailed Description

This cmdlet removes the specified DRS cluster groups from the cluster on which it resides.
