Remove-OrgVdc

Synopsis

This cmdlet removes organization virtual datacenters (vDCs).

Syntax

Remove-OrgVdc [-OrgVdc] <OrgVdc[]> [-Server <CIServer[]>] [-RunAsync] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-PipelineVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]

Related Commands

Detailed Description

This cmdlet removes organization virtual datacenters (vDCs).
