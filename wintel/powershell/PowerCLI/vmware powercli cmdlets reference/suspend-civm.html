Suspend-CIVM

Synopsis

This cmdlet suspends the specified virtual machine.

Syntax

Suspend-CIVM [-VM] <CIVM[]> [-RunAsync] [-Server <CIServer[]>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-PipelineVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]

Related Commands

Online version
Get-CIVM
Restart-CIVM
Start-CIVM

Detailed Description

This cmdlet suspends the specified virtual machine.

Notes

Examples

-------------- Example 1 --------------

Get-CIVM -Name 'MyVM' | Suspend-CIVM

Suspend the specified virtual machine.

-------------- Example 2 --------------

$task = (Get-CIVM -Name 'MyVM') | Suspend-CIVM -RunAsync

Suspends the specified virtual machine asynchronously.
