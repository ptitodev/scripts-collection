Get-OMRecommendation

Synopsis

This cmdlet retrieves recommendations from the specified vRealize Operations Manager server.

Syntax

Get-OMRecommendation [-Id <String[]>] [-Server <OMServer[]>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-PipelineVariable <String>] [<CommonParameters>]

Get-OMRecommendation -Alert <OMAlert[]> [-Server <OMServer[]>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-PipelineVariable <String>] [<CommonParameters>]

Get-OMRecommendation -AlertDefinition <OMAlertDefinition[]> [-Server <OMServer[]>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-PipelineVariable <String>] [<CommonParameters>]

Related Commands

Online version

Detailed Description

This cmdlet retrieves recommendations from the specified vRealize Operations Manager server.

Return Type

Notes

Examples

-------------- Example 1 --------------

Get-OMRecommendation

Lists all available recommendations.

-------------- Example 2 --------------

$alert = Get-OMAlert -Id <ID>
Get-OMRecommendation -Alert $alert

Retrieves all recommendations for a given alert.
