Get-ProxyServer

Synopsis

Get the proxy server list.

Syntax

Get-ProxyServer

Related Commands

Remove-ProxyServer
Add-ProxyServer

Detailed Description

Gets the proxy server list from the server.

You must have the "AutoDeploy.Profile.Edit" privilege on the root folder of vCenter Server to get the proxy server list.
