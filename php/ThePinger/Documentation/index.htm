Chapter 1 - Requirements

To run The Pinger, all you need is a server with PHP, mySQL and a Web Server application running. The system can be run on any Windows or Unix/Linux based system. If you are a macintosh owner and would like to see The Pinger work on Macintosh simply drop Shield Tech an e-mail by using the contact details found here. Then all you need is any standard web browser to control The Pinger.

 

Chapter 2 - Installation

To install The Pinger simply upload all the files to your web server and setup the access to config.php so that the install script can write the settings to the file. A CHMOD value of 0777 will be sufficient on a Unix/Linux based system, as for Windows systems they don't need a CHMOD value assigned.

Once this is done, simply point your web browser to the install.php script, this could be something like: http://yourdomain.com/thepinger/install.php. Simply enter all the details that the installation asks for, and when your done delete install.php from your web server to stop other people saving over your configuration. You can now goto the administration control panel and add systems to ping.
